01 L'interprète et ses deux modes

Python interagit avec la machine via un programme appelé interprète. On l'utilise de deux façons :

Mode interactif — un dialogue ligne par ligne. Les trois chevrons >>> sont l'invite de commandes : l'interprète attend une instruction et affiche aussitôt le résultat.

Mode programme — on écrit une suite d'instructions dans un fichier (le code source), puis on les fait exécuter d'un coup. C'est le mode qu'on utilise dans les cellules ci-dessous.

💡

Un IDE (environnement de développement) ajoute du confort : coloration syntaxique, numérotation des lignes, auto-complétion, débogueur…

05 Variables & affectation

      
Un résultat peut être mémorisé pour être réutilisé. Le symbole = est une opération d'affectation : « range la valeur de droite dans le nom de gauche ».

Noms autorisés vs recommandés

      
Un nom peut comporter plusieurs caractères, mais certaines règles s'imposent :

      
    
        
  • Interdit : commencer par un chiffre (4x), utiliser un mot-clé (def), affecter à une valeur (1 = 2) → SyntaxError.
    • 
        
  • Utiliser une variable non définie → NameError.
    • 
        
  • Recommandé : pas d'accents, tout en minuscules, mots séparés par _ (convention snake_case) → ex. vitesse_max.
    •

08 Formater du texte : les f-strings

      
Les f-strings (préfixe f) insèrent des valeurs directement dans une chaîne, entre accolades. Simple et lisible :

      

        
Python

        
        

      

      
On peut aussi préciser un format après : — alignement, largeur, décimales. (Exemple de référence, à tester dans un vrai interprète Python.)

      
nb = 42
print(f"Nombre : {nb:.2f}")   # 42.00  (2 décimales)
print(f"Nombre : {nb:09}")    # 000000042  (largeur 9)
print(f"{'centré':^20}")      # aligné au centre sur 20 caractères
