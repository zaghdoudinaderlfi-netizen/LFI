EX 1 Facile Les diviseurs d'un nombre

Demande un entier n et affiche tous ses diviseurs.

Idée : d est un diviseur de n si le reste n % d vaut 0. On teste tous les d de 1 à n.

Indice 1 Comprendre

d est un diviseur de n si la division tombe juste, c'est-à-dire si le reste n % d vaut 0. Il faut tester tous les d possibles.

Indice 2 Les briques

Une boucle for d in range(1, n+1): et, à l'intérieur, un test if n % d == 0: pour afficher d.

Python

    

    

      
      
🔒Les corrections seront disponibles quand ton professeur les activera.

      

        
Correction

        
n = int(input("n : "))
for d in range(1, n + 1):
    if n % d == 0:
        print(d, "est un diviseur de", n)

        
On combine une boucle (chapitre 2) et un test (chapitre 3) : pour chaque d, on vérifie si la division tombe juste (n % d == 0).

      

    

  


  
  

    
EX 2 Intermédiaire Nombre premier ?

    
Demande un entier n et indique s'il est premier (divisible uniquement par 1 et lui-même).

    
Idée : compter les diviseurs, ou utiliser un « drapeau » booléen est_premier qu'on met à False dès qu'on trouve un diviseur.

    

  
Indice 1 Comprendre & tester à la main
« Premier » = aucun diviseur entre 2 et n−1. Vérifie : 7 ne se divise par aucun → premier ; 9 se divise par 3 → pas premier.

  
Indice 2 La logique (drapeau)
je suppose que n EST premier
pour chaque d de 2 à n-1 :
    si d divise n :
        n N'EST PAS premier
afficher le verdict

  
Indice 3 Les briques
est_premier = True au départ ; une boucle for d in range(2, n): ; le test n % d == 0 baisse le drapeau ; à la fin if est_premier:.



    

      
Python

      
      

    

    

      
      
🔒Les corrections seront disponibles quand ton professeur les activera.

      

        
Correction

        
n = int(input("n : "))
est_premier = True
if n < 2:
    est_premier = False
for d in range(2, n):
    if n % d == 0:
        est_premier = False
if est_premier:
    print(n, "est premier")
else:
    print(n, "n'est pas premier")

        
Le drapeau booléen est_premier part à True et passe à False dès qu'un diviseur est trouvé. À la fin, on teste le drapeau. (Astuce : on peut s'arrêter à √n pour aller plus vite.)

      

    

  


  
  

    
EX 3 Intermédiaire Triangle : possible ? quel type ?

    
Demande trois longueurs et indique si elles forment un triangle. Si oui, précise s'il est équilatéral, isocèle ou scalène.

    
Inégalité triangulaire : chaque côté doit être plus petit que la somme des deux autres.

    

  
Indice 1 D'abord : est-ce un triangle ?
Inégalité triangulaire : chaque côté doit être plus petit que la somme des deux autres. Trois conditions reliées par and.

  
Indice 2 Ensuite : quel type ?
Si c'est un triangle (conditions imbriquées) : a == b == c → équilatéral ; deux côtés égaux → isocèle ; sinon → scalène.



    

      
Python

      
      

    

    

      
      
🔒Les corrections seront disponibles quand ton professeur les activera.

      

        
Correction

        
a = float(input("a : "))
b = float(input("b : "))
c = float(input("c : "))

if a < b + c and b < a + c and c < a + b:
    if a == b == c:
        print("Triangle équilatéral")
    elif a == b or b == c or a == c:
        print("Triangle isocèle")
    else:
        print("Triangle scalène")
else:
    print("Ce n'est pas un triangle")

        
On teste d'abord l'inégalité triangulaire (avec and), puis à l'intérieur (conditions imbriquées), on classe selon l'égalité des côtés. Note a == b == c (chaîne de comparaisons) pour l'équilatéral.

      

    

  


  
  

    
EX 4 Avancé Triangle rectangle ?

    
Enrichis l'exercice 3 : indique aussi si le triangle est rectangle (théorème de Pythagore : le carré du plus grand côté = somme des carrés des deux autres).

    

  
Indice 1 Le critère
Théorème de Pythagore : le triangle est rectangle si le carré du plus grand côté = somme des carrés des deux autres.

  
Indice 2 Les briques
Trouve le plus grand côté avec max(a,b,c). Compare son carré à la somme des carrés des deux autres.



    

      
Python

      
      

    

    

      
      
🔒Les corrections seront disponibles quand ton professeur les activera.

      

        
Correction

        
a = float(input("a : "))
b = float(input("b : "))
c = float(input("c : "))

if a < b + c and b < a + c and c < a + b:
    # on identifie le plus grand cote (hypoténuse potentielle)
    grand = max(a, b, c)
    autres = a + b + c - grand
    # somme des carrés des deux autres
    if grand**2 == (a**2 + b**2 + c**2 - grand**2):
        print("Triangle rectangle")
    else:
        print("Triangle non rectangle")
else:
    print("Ce n'est pas un triangle")

        
Astuce : a**2 + b**2 + c**2 - grand**2 donne la somme des carrés des deux autres côtés. On compare au carré du plus grand. (Avec des flottants, on tolère parfois une petite marge d'erreur.)

      

    

  


  
  

    
EX 5 Avancé Trois points alignés ?

    
Demande les coordonnées entières de trois points et indique s'ils sont alignés.

    
Critère sans division : les vecteurs AB et AC sont colinéaires si (xB−xA)·(yC−yA) − (yB−yA)·(xC−xA) == 0.

    

  
Indice 1 L'idée géométrique
Trois points sont alignés si les vecteurs AB et AC ont la même direction (colinéaires). On évite la division (pente) qui plante sur les droites verticales.

  
Indice 2 Le critère (produit en croix)
Calcule (xb-xa)*(yc-ya) - (yb-ya)*(xc-xa). S'il vaut 0 → alignés.



    

      
Python

      
      

    

    

      
      
🔒Les corrections seront disponibles quand ton professeur les activera.

      

        
Correction

        
xa = int(input("xA : ")); ya = int(input("yA : "))
xb = int(input("xB : ")); yb = int(input("yB : "))
xc = int(input("xC : ")); yc = int(input("yC : "))

produit_croise = (xb - xa) * (yc - ya) - (yb - ya) * (xc - xa)
if produit_croise == 0:
    print("Les points sont alignés")
else:
    print("Les points ne sont pas alignés")

        
On évite la division (et le piège des droites verticales) en utilisant le produit en croix des vecteurs AB et AC : nul ⟺ alignés.

      

    

  


  
  

    
EX 6 Facile Distance entre deux nombres

    
Demande deux nombres et affiche la distance entre eux (toujours positive).

    
La distance est la valeur absolue de la différence. À faire avec un test (sans abs) pour s'entraîner.

    

  
Indice 1 Comprendre
La distance entre deux nombres est toujours positive : c'est la valeur absolue de leur différence.

  
Indice 2 Les briques
Avec un test : if x > y: ... else: ... pour soustraire dans le bon sens. (Ou directement abs(x - y).)



    

      
Python

      
      

    

    

      
      
🔒Les corrections seront disponibles quand ton professeur les activera.

      

        
Correction

        
x = float(input("x : "))
y = float(input("y : "))

if x > y:
    distance = x - y
else:
    distance = y - x
print("Distance :", distance)

# Version courte avec la valeur absolue :
# print("Distance :", abs(x - y))

        
Le if/else garantit un résultat positif. La fonction abs() fait la même chose en une ligne.

      

    

  


  
  

    
EX 7 Intermédiaire Année bissextile

    
Demande une année et indique si elle est bissextile.

    
Règle : divisible par 4, sauf les années divisibles par 100 qui ne sont pas divisibles par 400.

    

  
Indice 1 La règle
Une année est bissextile si elle est divisible par 4, sauf les années divisibles par 100 qui ne le sont pas par 400.

  
Indice 2 En expression booléenne
Tout tient en une condition : (annee % 4 == 0 and annee % 100 != 0) or (annee % 400 == 0). Teste 2000, 1900, 2024.



    

      
Python

      
      

    

    

      
      
🔒Les corrections seront disponibles quand ton professeur les activera.

      

        
Correction

        
annee = int(input("Année : "))

if (annee % 4 == 0 and annee % 100 != 0) or (annee % 400 == 0):
    print(annee, "est bissextile")
else:
    print(annee, "n'est pas bissextile")

        
Toute la règle tient dans une expression booléenne combinant and et or. Teste 2000 (oui), 1900 (non), 2024 (oui).

      

    

  


  
  

    
EX 8 Intermédiaire FizzBuzz (1 à 100)

    
Parcours les entiers de 1 à 100. Affiche fizz pour les multiples de 3, buzz pour les multiples de 5, fizzbuzz pour les multiples de 15, sinon l'entier.

    
Le classique des entretiens ! Piège : tester 15 d'abord (sinon on n'y arrive jamais).

    

  
Indice 1 Fais 1 à 15 à la main
Note ce qui s'affiche : 1,2,fizz,4,buzz,fizz,7,8,fizz,buzz,11,fizz,13,14,fizzbuzz.

  
Indice 2 Le piège de l'ordre
Teste 15 en premier (avec elif), sinon un multiple de 15 afficherait « fizz » et n'arriverait jamais à « fizzbuzz ».



    

      
Python

      
      

    

    

      
      
🔒Les corrections seront disponibles quand ton professeur les activera.

      

        
Correction

        
for i in range(1, 101):
    if i % 15 == 0:
        print("fizzbuzz")
    elif i % 3 == 0:
        print("fizz")
    elif i % 5 == 0:
        print("buzz")
    else:
        print(i)

        
L'ordre est crucial : on teste 15 en premier. Avec elif, dès qu'une condition est vraie, les suivantes sont ignorées — c'est exactement ce qu'on veut.

      

    

  


  
  

    
EX 9 Intermédiaire Trois entiers dans l'ordre croissant

    
Demande trois entiers et affiche-les dans l'ordre croissant.

    

  
Indice 1 Repère les extrêmes
Le plus petit avec min(a,b,c), le plus grand avec max(a,b,c).

  
Indice 2 L'astuce pour le moyen
Le « moyen » = somme des trois − le plus petit − le plus grand. Puis affiche petit, moyen, grand.



    

      
Python

      
      

    

    

      
      
🔒Les corrections seront disponibles quand ton professeur les activera.

      

        
Correction

        
a = int(input("a : "))
b = int(input("b : "))
c = int(input("c : "))

petit = min(a, b, c)
grand = max(a, b, c)
moyen = a + b + c - petit - grand   # astuce : la somme moins les extrêmes
print(petit, moyen, grand)

        
Astuce élégante : le « moyen » est la somme des trois moins le plus petit et le plus grand. (On peut aussi le faire entièrement avec des if imbriqués.)

      

    

  


  
  

    
EX 10 Avancé Équation du second degré

    
Résous ax² + bx + c = 0 : selon le discriminant Δ, affiche 0, 1 ou 2 solutions.

    
Δ = b² − 4ac. Si Δ > 0 : deux solutions ; Δ = 0 : une ; Δ < 0 : aucune (réelle).

    

  
Indice 1 Rappelle le cours de maths
Δ = b² − 4ac. Trois cas : Δ > 0 (deux solutions), Δ = 0 (une), Δ < 0 (aucune réelle).

  
Indice 2 Les briques
Calcule delta, puis if/elif/else sur son signe. Utilise sqrt (module math) seulement quand Δ ≥ 0.



    

      
Python

      
      

    

    

      
      
🔒Les corrections seront disponibles quand ton professeur les activera.

      

        
Correction

        
from math import sqrt
a = float(input("a : "))
b = float(input("b : "))
c = float(input("c : "))

delta = b**2 - 4*a*c
if delta > 0:
    x1 = (-b - sqrt(delta)) / (2*a)
    x2 = (-b + sqrt(delta)) / (2*a)
    print("Deux solutions :", x1, "et", x2)
elif delta == 0:
    x = -b / (2*a)
    print("Une solution :", x)
else:
    print("Aucune solution réelle")

        
Trois cas selon le signe de Δ → trois branches if/elif/else. On calcule la racine seulement quand Δ ≥ 0 (sinon sqrt planterait).

      

    

  


  
  

    
EX 11 Avancé turtle — un damier n×n

    
Demande un entier n et trace un damier de côté n (cases noires et blanches en alternance).

    
Idée : deux boucles imbriquées (ligne, colonne). Une case est noire si (ligne + colonne) est pair.

    

  
Indice 1 La structure
Un damier = une grille. Il te faut deux boucles imbriquées : une pour les lignes, une pour les colonnes.

  
Indice 2 La couleur (le cœur)
Une case est noire ou blanche selon la parité : if (ligne + colonne) % 2 == 0: → une couleur, sinon l'autre. Puis dessine une case pleine.



    

      
Python · turtle

      
      

      

    

    

      
      
🔒Les corrections seront disponibles quand ton professeur les activera.

      

        
Correction

        
from turtle import *

n = int(input("Taille du damier : "))
cote = 30
speed(0)
for ligne in range(n):
    for colonne in range(n):
        # couleur selon la parité de (ligne + colonne)
        if (ligne + colonne) % 2 == 0:
            fillcolor("black")
        else:
            fillcolor("white")
        # on se place et on dessine une case pleine
        penup()
        goto(colonne * cote, ligne * cote)
        pendown()
        begin_fill()
        for _ in range(4):
            forward(cote)
            left(90)
        end_fill()

        
Le cœur est le test de parité (ligne + colonne) % 2 == 0 qui décide de la couleur — c'est ce qui crée l'alternance du damier. Le reste, ce sont deux boucles imbriquées (chapitre 2).
