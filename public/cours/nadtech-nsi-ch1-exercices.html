EX 1 Facile Priorité des opérations

Réécris les expressions suivantes en explicitant toutes les parenthèses :

1 + 2 * 3 - 4  ·  1 + 2 / 4 * 3  ·  1 - a + a * a / 2 - a * a * a / 6 + a * a * a * a / 24

Rappel : * et / sont prioritaires sur + et -. À priorité égale, on lit de gauche à droite. Exécute pour vérifier les valeurs :

Indice 1 Rappelle-toi la règle

Les opérations * et / passent avant + et -. À priorité égale, on lit de gauche à droite.

Indice 2 Comment faire

Souligne d'abord chaque * et / et mets-les entre parenthèses. Puis enchaîne les +/- restants, de gauche à droite, en parenthésant à chaque étape.

Python

    

    

      
      
🔒Les corrections seront disponibles quand ton professeur les activera.

      

        
Correction

        
1 + 2 * 3 - 4      →  ( (1 + (2 * 3)) - 4 )        = 3
1 + 2 / 4 * 3      →  ( 1 + ((2 / 4) * 3) )        = 2.5
1 - a + a*a/2 - a*a*a/6 + a*a*a*a/24
   →  ( ( ( (1 - a) + ((a*a)/2) ) - (((a*a)*a)/6) ) + ((((a*a)*a)*a)/24) )

        
Pour la 2ᵉ : / et * ont la même priorité, on lit de gauche à droite, donc 2/4 d'abord (= 0.5), puis ×3 (= 1.5).

        La 3ᵉ est le début du développement de e^(-a) : chaque terme est un produit divisé, additionné/soustrait de gauche à droite.

EX 3 Facile Aire d'un rectangle

    
Écris un programme qui demande les deux côtés (entiers) d'un rectangle et affiche son aire.

    
Rappel : input() renvoie une chaîne ; on la convertit avec int(). Complète puis exécute :

Indice 2 Les briques

Python

EX 4 Intermédiaire Conversion vers la base 10

    
Demande une base (entre 2 et 36) et un nombre, puis affiche ce nombre en base 10.

    Indice : int(chaine, base) convertit une chaîne écrite dans une base donnée.

    

  
Indice 1 L'outil donné
L'énoncé te donne int(chaine, base) : il convertit une chaîne écrite dans une base vers un entier base 10.

  
Indice 2 Attention au type
Le nombre saisi doit rester une chaîne (donc input() sans int()), car int(chaine, base) lit ses caractères.



    

      
Python

      
      

    

    

      
      
🔒Les corrections seront disponibles quand ton professeur les activera.

      

        
Correction

        
base = int(input("Base (entre 2 et 36) : "))
nombre = input("Nombre : ")
print(f"En base 10 : {int(nombre, base)}")

        
Le nombre saisi reste une chaîne (ex. "1010"), car int(chaine, base) lit ses caractères selon la base. Exemple : int("1010", 2) donne 10.

      

    

  


  
  

    
EX 5 Intermédiaire Secondes → heures, minutes, secondes

    
Demande un nombre de secondes et affiche-le sous la forme heures / minutes / secondes.

    
Clés : // (quotient) et % (reste). 1 h = 3600 s, 1 min = 60 s.

    

  
Indice 1 Fais-le à la main
Prends 3661 secondes. Combien d'heures entières ? Combien de minutes ensuite ? Combien de secondes restantes ?

  
Indice 2 Les briques
1 h = 3600 s, 1 min = 60 s. Utilise // (quotient) pour extraire et % (reste) pour ce qui reste.



    

      
Python

      
      

    

    

      
      
🔒Les corrections seront disponibles quand ton professeur les activera.

      

        
Correction

        
total = int(input("Nombre de secondes : "))
heures = total // 3600
minutes = (total % 3600) // 60
secondes = total % 60
print(f"{heures} h {minutes} min {secondes} s")

        
On retire d'abord les heures (// 3600), il reste total % 3600 secondes, dont on extrait les minutes, puis le reste donne les secondes.

EX 7 Avancé turtle — le drapeau français

    
Dessine un drapeau français (trois bandes : bleu, blanc, rouge).

    Utiles : fillcolor, begin_fill, end_fill.

    
Idée : pour chaque couleur, remplir un rectangle puis se décaler vers la droite.

    

  
Indice 1 Découpe le problème
Un drapeau = 3 bandes côte à côte (bleu, blanc, rouge). Tu fais 3 fois la même chose : remplir un rectangle, puis te décaler à droite.

  
Indice 2 Les briques
Une boucle sur ["blue","white","red"] ; pour chaque couleur : fillcolor + begin_fill() … end_fill() ; puis penup() et avancer de la largeur d'une bande.



    

      
Python · turtle

      
      

      

    

    

      
      
🔒Les corrections seront disponibles quand ton professeur les activera.

      

        
Correction

        
from turtle import *

speed(0)
pencolor("black")
L = 50    # largeur d'une bande
H = 100   # hauteur du drapeau

for c in ["blue", "white", "red"]:
    fillcolor(c)
    begin_fill()
    forward(L); left(90)
    forward(H); left(90)
    forward(L); left(90)
    forward(H); left(90)
    end_fill()
    penup(); forward(L); pendown()

        
On parcourt les trois couleurs. Pour chacune : on remplit un rectangle (begin_fill … end_fill), puis on lève le crayon et on se décale de la largeur d'une bande vers la droite.
