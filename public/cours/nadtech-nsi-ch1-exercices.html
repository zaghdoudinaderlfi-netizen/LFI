    Indice : int(chaine, base) convertit une chaîne écrite dans une base donnée.

    

      
Python

      
      

    

    

      
      
🔒Les corrections seront disponibles quand ton professeur les activera.

      

        
Correction

        
base = int(input("Base (entre 2 et 36) : "))
nombre = input("Nombre : ")
print(f"En base 10 : {int(nombre, base)}")

        
Le nombre saisi reste une chaîne (ex. "1010"), car int(chaine, base) lit ses caractères selon la base. Exemple : int("1010", 2) donne 10.

      

    

  


  
  

    
EX 5 Secondes → heures, minutes, secondes

    
Demande un nombre de secondes et affiche-le sous la forme heures / minutes / secondes.

    
Clés : // (quotient) et % (reste). 1 h = 3600 s, 1 min = 60 s.

    

      
Python

      
      

    

    

      
      
🔒Les corrections seront disponibles quand ton professeur les activera.

      

        
Correction

        
total = int(input("Nombre de secondes : "))
heures = total // 3600
minutes = (total % 3600) // 60
secondes = total % 60
print(f"{heures} h {minutes} min {secondes} s")

        
On retire d'abord les heures (// 3600), il reste total % 3600 secondes, dont on extrait les minutes, puis le reste donne les secondes.

EX 7 turtle — le drapeau français

    
Dessine un drapeau français (trois bandes : bleu, blanc, rouge).

    Utiles : fillcolor, begin_fill, end_fill.

    
Idée : pour chaque couleur, remplir un rectangle puis se décaler vers la droite.

    

      
Python · turtle

      
      

      

    

    

      
      
🔒Les corrections seront disponibles quand ton professeur les activera.

      

        
Correction

        
from turtle import *

speed(0)
pencolor("black")
L = 50    # largeur d'une bande
H = 100   # hauteur du drapeau

for c in ["blue", "white", "red"]:
    fillcolor(c)
    begin_fill()
    forward(L); left(90)
    forward(H); left(90)
    forward(L); left(90)
    forward(H); left(90)
    end_fill()
    penup(); forward(L); pendown()

        
On parcourt les trois couleurs. Pour chacune : on remplit un rectangle (begin_fill … end_fill), puis on lève le crayon et on se décale de la largeur d'une bande vers la droite.
