EX 1 Calculer 2ⁿ

Demande un entier n à l'utilisateur, puis calcule et affiche 2ⁿ (2 puissance n) — avec une boucle, en partant d'un accumulateur.

Idée : pars de resultat = 1 et multiplie par 2 à chaque tour.

Python

    

    

      
      
🔒Les corrections seront disponibles quand ton professeur les activera.

      

        
Correction

        
n = int(input("n : "))
resultat = 1
for _ in range(n):
    resultat = resultat * 2
print(f"2^{n} = {resultat}")

        
L'accumulateur resultat part de 1 et double à chaque tour : 1 → 2 → 4 → 8… (Python sait aussi faire 2 ** n, mais ici on s'entraîne avec la boucle.)

      

    

  


  
  

    
EX 2 Le produit 1 × 2 × … × 100

    
Calcule et affiche 1 × 2 × … × 100 (la factorielle de 100).

    
Astuce : range(1, 101) donne les entiers de 1 à 100. Accumulateur initialisé à 1.

    

      
Python

      
      

    

    

      
      
🔒Les corrections seront disponibles quand ton professeur les activera.

      

        
Correction

        
produit = 1
for i in range(1, 101):
    produit = produit * i
print(produit)

        
On parcourt i de 1 à 100 (101 exclu). Python gère les très grands entiers : le résultat a 158 chiffres !

EX 4 Intérêts d'un livret

    
Demande une somme initiale s, un taux annuel t (en %) et un nombre d'années n. Affiche les intérêts perçus chaque année et le montant total après n années.

    
Chaque année : intérêt = montant × t / 100, puis montant = montant + intérêt.

    

      
Python

      
      

    

    

      
      
🔒Les corrections seront disponibles quand ton professeur les activera.

      

        
Correction

        
s = float(input("Somme initiale : "))
t = float(input("Taux annuel (%) : "))
n = int(input("Nombre d'années : "))

montant = s
for annee in range(1, n + 1):
    interet = montant * t / 100
    montant = montant + interet
    print(f"Année {annee} : intérêts = {interet:.2f}, total = {montant:.2f}")

print(f"Montant après {n} ans : {montant:.2f}")

        
C'est un accumulateur (montant) : chaque année repart du total précédent — ce sont les intérêts composés. Le :.2f arrondit à 2 décimales.

      

    

  


  
  

    
EX 5 Moyenne pondérée

    
Demande un nombre de notes n, puis pour chaque note : la note et son coefficient. Affiche la moyenne pondérée.

    
Moyenne pondérée = (somme des note×coef) ÷ (somme des coef). Deux accumulateurs.

    

      
Python

      
      

    

    

      
      
🔒Les corrections seront disponibles quand ton professeur les activera.

      

        
Correction

        
n = int(input("Nombre de notes : "))
somme_notes = 0
somme_coefs = 0
for i in range(n):
    note = float(input(f"Note {i + 1} : "))
    coef = float(input(f"Coefficient {i + 1} : "))
    somme_notes = somme_notes + note * coef
    somme_coefs = somme_coefs + coef
moyenne = somme_notes / somme_coefs
print(f"Moyenne pondérée : {moyenne:.2f}")

        
On accumule séparément le total pondéré (note×coef) et le total des coefficients, puis on divise.

      

    

  


  
  

    
EX 6 Former un nombre avec n chiffres

    
Demande un nombre de chiffres n, puis n chiffres un par un, et reconstitue le nombre formé par ces chiffres dans l'ordre.

    
Astuce classique : à chaque chiffre, nombre = nombre * 10 + chiffre.

    

      
Python

      
      

    

    

      
      
🔒Les corrections seront disponibles quand ton professeur les activera.

      

        
Correction

        
n = int(input("Combien de chiffres ? "))
nombre = 0
for i in range(n):
    chiffre = int(input(f"Chiffre {i + 1} : "))
    nombre = nombre * 10 + chiffre
print(f"Nombre formé : {nombre}")

        
À chaque tour, on « décale » le nombre d'un rang vers la gauche (×10) et on ajoute le nouveau chiffre. Ex : 3 puis 7 puis 2 → 0→3→37→372.

      

    

  


  
  

    
EX 7 turtle — polygone régulier à n côtés

    
Demande un nombre de côtés n, puis trace avec turtle un polygone régulier à n côtés.

    
Clé : pour un polygone régulier à n côtés, l'angle extérieur vaut 360 / n.

    

      
Python · turtle

      
      

      

    

    

      
      
🔒Les corrections seront disponibles quand ton professeur les activera.

      

        
Correction

        
from turtle import *

n = int(input("Nombre de côtés : "))
for _ in range(n):
    forward(100)
    left(360 / n)

        
Le même code marche pour tout n : triangle (n=3), carré (n=4), pentagone (n=5)… Plus n est grand, plus le polygone ressemble à un cercle.
